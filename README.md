# go filesserver

Run fs.exe
```javascript
   http://localhost:6200
   http://192.168.1.56:6200
```

cmd
```javascript
   $ fs.exe 6790
   http://localhost:6790
   http://192.168.1.56:6790
```

Add the file's directory to Path environment variable
```javascript
   $ fs 7956
   http://localhost:7956
   http://192.168.1.56:7956
```
